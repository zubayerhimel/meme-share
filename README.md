## Meme-share application with MERN stack

### Backend
- NodeJS
- Express
- MongoDB
- Twilio sms system for Forgot Password
- API
   - Authentication
      - Normal auth
- Profile
  - Get profile
  - Forgot password with OTP verification (Twilio)
- Meme Post 
  - Create post 
  - Get All posts
  - Get each post
  - Image upload on Cloudinary
- Comments
  - Create comments
  - Delete comments
- Like Posts
- UnLike Posts

### Frontend
- ReactJS
- React router
- Axios
- Material-UI
- Local storage
