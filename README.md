## Meme-share application with MERN stack

### To run  the project
1. `cd meme-sharer`
2. `npm  i`
3. `cd client`
4. `npm i`
5. `cd ..`
6. `npm run dev`
7. Project will run on `PORT 3000`

### Backend
- NodeJS
- Express
- MongoDB
- Twilio sms system for Forgot Password
- API
   - Authentication
      - Normal auth
- Profile
  - Get profile
  - Forgot password with OTP verification (Twilio)
- Meme Post 
  - Create post 
  - Get All posts
  - Get each post
  - Image upload on Cloudinary
- Comments
  - Create comments
  - Delete comments
- Like Posts
- UnLike Posts

### Frontend
- ReactJS
- React router
- Axios
- Material-UI
- Local storage
